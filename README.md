 Portfolio

A modern, minimalist portfolio built with a focus on elegant typography, smooth motion, and a clean "white-on-white" aesthetic.

## 🚀 Built With

- **React + Vite**: For a fast, modern development experience.
- **Tailwind CSS**: Leveraging the latest styling patterns for a high-end feel.
- **Framer Motion**: Powering all animations, including the multilingual hero greeting and the interactive scroll progress tracker.
- **Lucide React**: Clean, consistent iconography.
- **Instrument Serif & Poppins**: Curated typography for a sophisticated look.

## 🛠️ Getting Started

1. **Clone the repo**:
   ```bash
   git clone https://github.com/vrindaatalwar/portfolio.git
   ```
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Run locally**:
   ```bash
   npm run dev
   ```
